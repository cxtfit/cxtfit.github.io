Q&A

The most often encountered problem occurs when opening an Excel 2003 file in Excel 2007. A Microsoft Visual Basic dialog pops up with a Compile error: Can't find project or library. We recommend using Windows Task Manager to end Excel. Please refer to Section 11 of the tutorial for more discussion about Excel 2007.

You may also avoid this by using the Excel files in the folder solver-not-referenced.

For Mac OS, you may click menu Tools -> Add-ins, check solver.xlam, click OK to install Solver. Click menu Tools -> Macro -> Visual Basic Editor to open the VBA Editor. Then click menu Tools -> References to open the reference dialog. Click on Solver.
